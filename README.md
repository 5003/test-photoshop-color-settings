# カラー設定

## 説明

- Japan Web Internet（Web・インターネット用）

    インターネットホームページに表示するコンテンツなど、印刷用ではないコンテンツを日本において作成するときに適したカラー設定です。RGB コンテンツは sRGB に変換されます。

- Japan Prepress（プリプレス用）

    日本における一般的な印刷条件に合わせたコンテンツを作成するときに適したカラー設定です。CMYK の値は保持され、カラープロファイルについての警告は必要に応じて表示されます。

- Japan General  Purpose（一般用）

    日本における画面表示用および印刷用のコンテンツを作成するときの一般用のカラー設定です。カラープロファイルについての警告は表示されません。

- Japan Magazine（雑誌広告用）

    日本における雑誌広告を作成するときに適したカラー設定です。この設定は社団法人日本雑誌協会の DDCP を出力ターゲットにした JMPA カラー (雑誌広告基準カラー) を参考に作成されたプロファイルに対応しています。 CMYK の値は保持され、カラープロファイルについての警告は必要に応じて表示されます。

- Japan Newspaper（新聞用）

    日本の新聞印刷のための環境設定です。CMYKの値は保持され、カラープロファイルについての警告は必要に応じて表示されます。
